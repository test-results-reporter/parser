# test-results-parser

Parse test results from JUnit, TestNG, xUnit, Mocha(json), Cucumber(json) and many more

## Support

| Result Type                   | Support |
|-------------------------------|---------|
| TestNG                        | ✅      |
| JUnit                         | ✅      |
| NUnit (v2 & v3)               | ✅      |
| xUnit                         | ✅      |
| Mocha (json & mochawesome)    | ✅      |
| Cucumber                      | ✅      |
